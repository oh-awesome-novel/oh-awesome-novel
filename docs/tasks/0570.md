# 0570 Workspace Global Search

> Status: Planned
> Milestone: M8 Minimal Copilot Interface

## Goal

在 workspace 工作台中增加全局搜索能力，覆盖当前小说 workspace 的 Markdown / YAML / 普通文本文件。

搜索引擎使用 [`lucaong/minisearch`](https://github.com/lucaong/minisearch)。MiniSearch 是 JavaScript 内存全文搜索引擎，可在 Node 和浏览器运行，支持前缀搜索、模糊搜索、字段 boosting、结果排序和自动建议。项目只使用它做本地 workspace 搜索，不引入远程搜索服务或向量数据库。

## User Experience

用户在 workspace 内可以打开全局搜索入口，输入关键词后看到匹配文件列表。点击搜索结果后：

- 左侧文件树选中对应文件。
- 中间纯文本查看器打开对应文件。
- 如果可行，滚动到首个匹配位置并高亮匹配片段。
- 右侧 Copilot 不应因搜索自动打开。

搜索入口可以是顶部命令、侧栏按钮或快捷键触发的弹窗。第一版不要求复杂查询语言。

## Search Scope

默认索引：

- `characters/`
- `world/`
- `chapters/`
- `state/`
- `timeline/`
- `foreshadow/`
- `summaries/`
- `.oan/constitution/`
- `.oan/workflow.yaml`

默认排除：

- `.git/`
- `.workspace/`
- `.oan/sessions/`
- `node_modules/`
- 大型二进制文件。
- 非当前 workspace 的文件。

## Index Shape

每个可搜索文档至少包含：

- `id`: workspace-relative file path。
- `path`: workspace-relative file path。
- `name`: 文件名。
- `domain`: 从路径推断的小说领域，例如 character / world / chapter / state。
- `content`: 原始文本内容。

第一版可以只索引纯文本内容和路径。后续可加入 frontmatter、标题、章节号、角色 id 等字段 boosting。

## Deliverables

- MiniSearch dependency decision and installation plan.
- Workspace search index builder.
- Read-only workspace file scanner for searchable text files.
- Search API or local backend endpoint.
- Search UI entry.
- Search result list with file path, domain, and snippet.
- Click result to open the plain-text file viewer.
- Empty result state.
- Index rebuild / refresh strategy.
- Tests for scope filtering and hidden directory exclusion.

## Done Criteria

- 用户可以在 workspace 内搜索当前小说工程。
- 搜索结果来自 MiniSearch index。
- 搜索不读取 workspace 外部路径。
- 搜索不会索引 `.git/`、`.workspace/` 或 `.oan/sessions/`。
- 搜索结果点击后打开对应文件的纯文本视图。
- 未选择文件的 workspace home 状态不会被搜索破坏，除非用户明确打开某个结果。
- 搜索不会自动触发 Copilot 或写入工具。
- 索引可以在 workspace 切换时重建。

## Constraints

- 不引入 Elasticsearch、SQLite FTS、LanceDB、向量数据库或远程搜索服务。
- MiniSearch index 是派生缓存，不是事实源。
- 不把搜索结果默认塞进 agent context。
- 不让 frontend 绕过 backend / agent 直接访问 filesystem。
- 不索引隐藏目录，除非后续任务明确允许某个内部只读范围。
- 不做 Markdown / YAML 渲染或手动编辑。
