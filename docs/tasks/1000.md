## Goal

把 `OAN_AGENT_WRITING_GUIDE_REFERENCE_NOTES.md` 中已经确认的写作指引方向，收敛为稳定 Spec、core 类型和 `novel-copilot` skill contract。

- `packages/core/src/novel-copilot-skill.ts` 更新：

  - 增加 vNext prompt sections

## Done Criteria

- `novel-copilot` 明确区分 Writing Mode、Review、Settle、Play、Reference use。
- `/审稿` 不再被描述为默认 settlement 触发点。
- `/规划大纲` 和 `/规划下一卷` 作为 planned quick commands 出现在 registry 中。
- capability id 可被 agent / session artifact 引用。
- Prompt 仍保持 Aider-style 单 agent tool loop，不引入 planner 或 multi-agent runtime。

## Constraints

- Skill 只包含 prompt / command / allowed tool contract，不包含业务逻辑。
- 不新增第二套 Tool 抽象。
- 不让 quick command 直接写真实目标文件。
- 不复制参考项目 prompt 原文。
