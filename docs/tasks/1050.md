# 1050 Projections And Project Health

> Status: Completed
> Milestone: M9 Summary And Memory Layer

## Related Plans

- [Projections And Project Health](../superpowers/plans/2026-06-19-projections-project-health.md)

## Goal

实现作者可读的派生 projection 和只读 project health panel，让状态、伏笔、时间线和摘要更容易被作者与 agent 检查。

## Deliverables

- projection generators：
  - state current view
  - foreshadow ledger
  - timeline view
  - progress summary
  - optional context snapshot
- `.oan/indexes/` 派生目录约定
- read-only project health model
- backend read endpoint 或 core helper
- UI 初版 Inspector / Dashboard
- tests 覆盖 projection 可删除重建

## Done Criteria

- Projection 可由 Object File Tree 重建。
- Project health 能展示缺失角色卡、未整理章节、active hooks、最新状态过期、时间线断层、未处理 PendingAction。
- Dashboard / Inspector 不承担事实写入。
- Projection 失败不应阻断普通写作链路第一版。

## Constraints

- 不使用 SQLite / vector read-model 作为核心状态层。
- 不让 projection 成为事实源。
- 不引入复杂 projection chain 失败阻断机制。

## Implementation Notes

- `packages/core/src/projections.ts` 已实现 `.oan/indexes/state.md`、`foreshadow.md`、`timeline.md`、`progress.md`、`context-snapshot.md` 的派生 Markdown generator 与 rebuild helper。
- `packages/core/src/project-health.ts` 已实现只读 project health model，覆盖缺失角色卡基本文件、未摘要章节、active hooks、latest state stale、timeline gaps 和 PendingAction count。
- `packages/backend/src/index.ts` 已增加只读 `GET /api/workspace/project-health` endpoint。
- `apps/desktop-ui/src/components/workspace/WorkspaceHome.vue` 已增加只读 Project health 摘要；`WorkspaceShell.vue` 和 `useWorkspaceApi.ts` 已接入 health 加载。
- `packages/core/src/index.ts` 已导出 projection / project health 类型与 helper。
- Projection 输出包含 generated warning，可删除后从 Object File Tree 重建；project health 不写事实文件。
- 已验证：
  - `npm run test:run --workspace @oh-awesome-novel/test-core`
