# 1060 Play Mode And Tavern Character Import

> Status: Completed
> Milestone: Play Mode

## Related Plans

- [Play Mode And Tavern Character Import](../superpowers/plans/2026-06-19-play-mode-tavern-import.md)

## Goal

规划并实现独立 Play Mode / Roleplay Sandbox，并接入 Tavern-compatible Character Card 导入能力。

Play Mode 既服务沉浸式体验，也能把 Play observation 转成写作草稿或参考，但默认不改 canonical truth。

## Deliverables

- Play Mode product spec update
- Play session filesystem spec：
  - transcript
  - play-local state
  - activated sources
  - observations
  - adoption candidates
- single world referee + multi-character voice/state modules design
- Tavern-compatible card parser implementation plan
- OAN character card interaction hints / lorebook integration
- UI route / panel plan
- tests 覆盖 import normalization、prompt injection guard、Play adoption PendingAction

## Done Criteria

- Play session 与 Writing Mode 并列，不只是 `/写下一章` 的隐藏 rehearsal。
- Play 读取角色卡、interaction hints、lorebook、世界规则、当前状态和起点。
- Play 结果默认只进入 Play session。
- 用户确认后，Play observation 才能转成章节草稿、状态、时间线或伏笔 PendingAction。
- Tavern-compatible import 不复制 SillyTavern AGPL 代码，不声称官方兼容。

## Constraints

- 不引入默认重型多 agent runtime。
- 不自动把 imported character prompt 覆盖 constitution / world / state。
- 不从分享站点自动抓取角色卡。
