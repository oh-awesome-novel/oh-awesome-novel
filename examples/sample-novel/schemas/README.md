# Schema 目录

此目录用于存放 YAML/Markdown 文件的 JSON Schema 或 Zod Schema 定义。

当前为空目录，后续将添加各领域文件的验证 Schema。

计划中的 Schema 列表：

- `character-meta.schema.json` — 角色 meta.yaml 验证
- `workflow.schema.json` — workflow.yaml 验证
- `state.schema.json` — state YAML 验证
- `timeline-event.schema.json` — 时间线事件验证
- `foreshadow.schema.json` — 伏笔验证
- `chapter-frontmatter.schema.json` — 章节 frontmatter 验证
